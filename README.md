# Indiansky Blog

**Indiansky Blog** is created to showcase the following:

1. Tailwind use-case
2. Responsive design
3. Creating a Blog

Currently, the site is designed for smartphone screens only. In subsequent integrations, the design will be updated for widescreen view as well.
